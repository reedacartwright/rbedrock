# Cast a value to a specific type

This is a helper function used (mostly) for modifying NBT values.

## Usage

``` r
rac_cast(x, to, ..., x_arg = "x")
```

## Arguments

- x:

  Value to cast

- to:

  Type to cast to

- ...:

  Currently unused.

- x_arg:

  Argument name for `x`, used in error messages.
